# rn-login-and-imgs

##Instructions to run the project in Windows

1. git clone https://github.com/Vincetroid/rn-login-and-imgs.git
2. npm i
3. Open the project through Android Studio by selecting the /android folder
4. Run it either with an Emulator, inside your other IDE run: npx react-native start
  or 
5. A device connnected via USB, inside your other IDE run: npx react-native run-android
